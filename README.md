# Sales Data Analysis Project

## 📊 Overview
This project analyzes sales data to identify trends, top-performing products, and revenue patterns.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📈 Key Insights
- Sales increased during peak seasons
- Top 5 products contribute major revenue
- Technology category performs best

## 🚀 How to Run
1. Open Jupyter Notebook
2. Run all cells in `Sales_Analysis.ipynb`

## 📂 Dataset
Superstore Sales Dataset (Kaggle)
